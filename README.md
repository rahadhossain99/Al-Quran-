## ⚙️ How to Run This Project Locally

### 📋 Prerequisites
Ensure you have [Android Studio](https://developer.android.com/studio) installed.

### 🛠️ Setup Steps

1. **Open Project:**
   Open Android Studio, click on **Open**, and choose the directory of this project.

2. **Sync Project:**
   Wait for Android Studio to sync the Gradle files and download necessary libraries.

3. **Configure Environment File:**
   Create a file named `.env` in the root folder of this project and add your API key:
